Características Principales

Corrección Inteligente

Utiliza el modelo qwen-turbo de DashScope para corregir errores ortográficos, gramaticales y de puntuación, optimizando el estilo para mejor legibilidad.

Preservación de Citas

Ignora inteligentemente el texto entre comillas para preservar citas literales, respetando el contenido original cuando es necesario.

Preservación del Formato

Mantiene intactos los elementos de formato del documento original, como negritas, cursivas, subrayados y colores.

Reglas Específicas para Español

Aplica reglas de capitalización específicas para títulos en español y maneja correctamente los dos puntos según normas gramaticales.

Versión de Prueba

Procesa documentos de hasta 150,000 caracteres en la versión de prueba, con notificación clara cuando se excede este límite.

Interfaz Intuitiva

Sube un archivo .docx, selecciona el idioma y obtén un documento corregido listo para descargar con solo un clic.

Revisar Cambios

Para una revisión detallada, utiliza la función "Comparar" de Microsoft Word:

  1. Abre Microsoft Word
  2. Ve a la pestaña "Revisar"
  3. Haz clic en "Comparar"
  4. Selecciona el documento original y el documento corregido
  5. Haz clic en "Aceptar" para ver todos los cambios realizados
